Запуск:
- Запустите локальный HTTP-сервер в папке проекта (например: python -m http.server 8000).
- Откройте http://localhost:8000 в браузере.

Команды:
- старт <имя>
- идти <направление> (north, south, east, west)
- осмотреть
- взять <предмет>
- использовать <предмет>
- говорить <имя>
- сражаться
- сохранить
- загрузить
- история
